# Estudio Fotográfico – Gestión de Sesiones

Este proyecto corresponde al desarrollo de un módulo web para la gestión de sesiones fotográficas, realizado como parte de la asignatura Desarrollo en Plataformas.

El sistema permite registrar, consultar, actualizar y eliminar sesiones fotográficas de manera digital, reemplazando el uso de agendas físicas.

## Caso desarrollado
Caso 25 – Estudio Fotográfico

El sistema permite gestionar sesiones como bodas, quinceañeras y sesiones para bebés, registrando la siguiente información:
- Tipo de sesión
- Nombre del cliente
- Teléfono
- Ubicación
- Fecha y hora
- Estado de la sesión (agendada, realizada, fotos entregadas, cancelada)

El sistema está diseñado para ser utilizado tanto en computadoras como en dispositivos móviles.

## Tecnologías utilizadas
- Laravel
- PHP
- PostgreSQL
- Bootstrap 5
- Laravel Herd
- Git y GitHub

## Funcionalidades
El sistema cuenta con un CRUD completo que permite:
- Crear sesiones fotográficas
- Listar sesiones registradas
- Editar la información de una sesión
- Eliminar sesiones

Incluye validaciones básicas, mensajes de confirmación y un diseño responsive.

## Base de datos
La estructura de la base de datos se gestiona mediante migraciones de Laravel, lo que permite crear las tablas directamente desde el framework sin necesidad de hacerlo manualmente. Esto facilita la organización y el control de cambios en la base de datos.
